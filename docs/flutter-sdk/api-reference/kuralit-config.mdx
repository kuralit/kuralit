Configuration class for initializing the Kuralit Flutter SDK.

## Class

<ParamField path="class" type="KuralitConfig">
  Configuration for the Kuralit SDK.
</ParamField>

## Constructor

<ParamField path="constructor" type="KuralitConfig()">
  Create a new KuralitConfig instance.

## Parameters

<ParamField path="serverUrl" type="String" required>
  WebSocket server URL (e.g., "wss://api.kuralit.com/ws").
</ParamField>

<ParamField path="apiKey" type="String" required>
  API key for authentication.
</ParamField>

<ParamField path="appId" type="String" required>
  Application identifier.
</ParamField>

<ParamField path="debug" type="bool" default="false">
  Enable debug logging.
</ParamField>

<ParamField path="reconnectEnabled" type="bool" default="true">
  Enable automatic reconnection.
</ParamField>

<ParamField path="maxReconnectAttempts" type="int" default="10">
  Maximum number of reconnection attempts.
</ParamField>

<ParamField path="reconnectDelayMs" type="int" default="1000">
  Initial reconnect delay in milliseconds.
</ParamField>

<ParamField path="heartbeatIntervalMs" type="int" default="30000">
  Heartbeat interval in milliseconds (0 to disable).
</ParamField>

## Usage Examples

### Basic Configuration

<CodeGroup>
```dart
final config = KuralitConfig(
  serverUrl: 'ws://localhost:8000/ws',
  apiKey: 'demo-api-key',
  appId: 'my-app',
  debug: true,
);

Kuralit.init(config);
```
</CodeGroup>

### Production Configuration

<CodeGroup>
```dart
final config = KuralitConfig.production(
  serverUrl: 'wss://api.example.com/ws',
  apiKey: 'prod-api-key',
  appId: 'my-app',
);

Kuralit.init(config);
```
</CodeGroup>

### Custom Reconnection Settings

<CodeGroup>
```dart
final config = KuralitConfig(
  serverUrl: 'ws://localhost:8000/ws',
  apiKey: 'demo-api-key',
  appId: 'my-app',
  reconnectEnabled: true,
  maxReconnectAttempts: 5,
  reconnectDelayMs: 2000,
  heartbeatIntervalMs: 30000,
);

Kuralit.init(config);
```
</CodeGroup>

## Adding Tools to AgentSession

Add tools when creating your agent:

<CodeGroup>
```python
from kuralit.server.agent_session import AgentSession
from kuralit.tools import Toolkit

# Define tools
def get_weather(location: str) -> str:
    """Get weather for a location."""
    return f"Weather in {location}: sunny, 22°C"

# Create toolkit
weather_tools = Toolkit(tools=[get_weather])

# Add to agent
agent = AgentSession(
    stt="deepgram/nova-2:en-US",
    llm="gemini/gemini-2.0-flash-001",
    tools=[weather_tools],  # Add tools here
    # ...
)
```
</CodeGroup>

## Creating Tools

### From Python Functions

<CodeGroup>
```python
from kuralit.tools.function import Function

def calculate(expression: str) -> str:
    """Calculate a mathematical expression."""
    try:
        result = eval(expression)
        return str(result)
    except Exception as e:
        return f"Error: {str(e)}"

# Convert to tool
tool = Function.from_callable(calculate)
```
</CodeGroup>

### Function Requirements

Functions must have:
- **Type hints** - For all parameters and return type
- **Docstrings** - Description of what the function does

## Using Toolkits

### Creating Toolkits

<CodeGroup>
```python
from kuralit.tools.toolkit import Toolkit

# Multiple related tools
def get_weather(location: str) -> str:
    """Get weather for a location."""
    return f"Weather in {location}: sunny, 22°C"

def get_forecast(location: str, days: int) -> str:
    """Get weather forecast."""
    return f"Forecast for {location}: {days} days of sunny weather"

# Create toolkit
weather_tools = Toolkit(
    name="weather",
    tools=[get_weather, get_forecast],
    instructions="Weather tools for getting current conditions and forecasts"
)
```
</CodeGroup>

### Adding Multiple Toolkits

<CodeGroup>
```python
calendar_tools = Toolkit(name="calendar", tools=[...])
weather_tools = Toolkit(name="weather", tools=[...])
reminder_tools = Toolkit(name="reminders", tools=[...])

agent = AgentSession(
    tools=[calendar_tools, weather_tools, reminder_tools],
    # ...
)
```
</CodeGroup>

## Complete Example

<CodeGroup>
```python
import os
from kuralit.server.agent_session import AgentSession
from kuralit.server.websocket_server import create_app
from kuralit.tools import Toolkit
import uvicorn

# Define tools
def get_weather(location: str) -> str:
    """Get weather for a location."""
    return f"Weather in {location}: sunny, 22°C"

def calculate(expression: str) -> str:
    """Calculate a mathematical expression."""
    try:
        result = eval(expression)
        return str(result)
    except Exception as e:
        return f"Error: {str(e)}"

# Create toolkit
utility_tools = Toolkit(
    name="utilities",
    tools=[get_weather, calculate],
    instructions="Utility tools for weather and calculations"
)

# Create agent with tools
agent = AgentSession(
    stt="deepgram/nova-2:en-US",
    llm="gemini/gemini-2.0-flash-001",
    vad="silero/v3",
    turn_detection="multilingual/v1",
    tools=[utility_tools],
    instructions="You are a helpful assistant with utility tools.",
)

# Create server
app = create_app(
    api_key_validator=lambda key: key == os.getenv("KURALIT_API_KEY", "demo-api-key"),
    agent_session=agent,
)

# Run
if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8000)
```
</CodeGroup>

## Python SDK-Specific Features

### Function Registration

<CodeGroup>
```python
from kuralit.tools.function import Function

# Manual function creation
tool = Function(
    name="my_function",
    description="Does something useful",
    parameters={
        "type": "object",
        "properties": {
            "param": {"type": "string"}
        },
        "required": ["param"]
    },
    entrypoint=my_function
)
```
</CodeGroup>

<CodeGroup>

```python
toolkit = Toolkit(

)
```
</CodeGroup>

## Best Practices

### Python SDK Best Practices

- **Use type hints** - Required for automatic tool generation
- **Write clear docstrings** - Help the agent understand your tools
- **Handle errors** - Return helpful error messages from tools
- **Group related tools** - Use toolkits to organize functionality
